UniqueRunner

public class UniqueRunner

A class used to run a function only once.

Example:

   let ur = UniqueRunner()
   var result = ur.runOnce(addNumbers(2, 2))
   result = ur.runOnce(addNumbers(5, 5))
   result = ur.runOnce(addNumbers(10, 10))

   result == 4 // true
